# **School Admin**

> **Descrição**: Plataforma web desenvolvido com React e Nextjs, que utiliza tecnologias modernas para criar uma aplicação ágil, robusta e escalável, para gerenciamento de instituições de ensino.

---

## **Tecnologias Utilizadas**

Este projeto foi construído com as seguintes tecnologias:

- **React**: Biblioteca para criação de interfaces de usuário.
- **TypeScript**: Superset do JavaScript com suporte a tipagem estática.
- **Node.js**: Ambiente de execução para JavaScript no lado do servidor.
- **Tailwind CSS**: Framework CSS baseado em classes utilitárias para estilização rápida.
- **React Query**: Biblioteca para gerenciamento de dados assíncronos e cache.
- **Shadcn**: Biblioteca de componentes de interface de usuário (UI) desenvolvida para React.
- **Zustand**: Biblioteca leve de gerenciamento de estado para aplicações React..

---

## **Como Utilizar**

Siga os passos abaixo para executar o projeto em sua máquina local:

### **1. Clone o Repositório**

No terminal, execute:
```bash
git clone https://github.com/Yhanko/school-frontend.git
cd school-frontend
```

### **2. Instale as Dependências**

Certifique-se de ter o Node.js instalado. Então, execute:
```bash
npm install
```

### **3. Inicie o Servidor de Desenvolvimento**

Para rodar a aplicação no modo de desenvolvimento, execute:
```bash
npm run dev
```
